# Новое русское вино

Сайт магазина авторского вина **«Новое русское вино»**.  
Скрипт генерирует HTML-страницу на основе Excel-файла с продукцией и запускает локальный веб-сервер.

Проект учебный, используется для практики Python, Jinja2 и работы с данными.

## Как установить

1) Скачайте архив с проектом нажав Code -> Download Zip, как на примере ниже и распакуйте его в нужной директории.

<img width="850" height="544" alt="image" src="https://github.com/user-attachments/assets/7317bef4-6ea6-4389-b2dc-ad84fd1205d1" />

2) Перед началом работы необходимо создать виртуальное окружение и установить зависимости.
Работа скрипта расчитана только для версии Python 3.10 и выше. В консоли `cmd` вводим:

```bash
path/to/your_project/python -m venv venv
venv\Scripts\activate
```

3) Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

## Запуск проекта

1) Базовый запуск (с файлом по умолчанию)

В проекте по умолчанию используется файл wine.xlsx, лежащий в корне проекта.

```bash
python main.py
```

2. Запуск с указанием своего файла данных

Можно передать путь к Excel-файлу с данными о винах через аргумент командной строки:

```bash
python main.py --wine_database path/to/your_wine.xlsx
```

## Проверка результата

После запуска:

1) В консоли нет ошибок
2) В корне проекта появляется файл:
```diff
index.html
```
3) В браузере откройте:
```cpp
http://127.0.0.1:8000
```
4) На странице:

- вина сгруппированы по категориям: «Белые вина», «Красные вина», «Напитки»
- отображаются названия, сорта, цены
- товары с акцией помечены значком
- в блоке «проверено временем» корректно отображается возраст компании
